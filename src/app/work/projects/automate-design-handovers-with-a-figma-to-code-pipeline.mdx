## Overview

This project involved building a fully responsive website from a Figma design. The goal was to translate the design accurately into clean, production-ready code using React and Next.js.

## Key Features

- **Pixel-Perfect UI**: Accurately translated Figma designs into responsive React components.
- **Responsive Design**: Ensured the website looks great on all screen sizes.
- **Modern Styling**: Used Tailwind CSS for clean and maintainable styles.

## Technologies Used

- **React and Next.js**: For building the frontend.
- **Tailwind CSS**: For styling and responsive layout.
- **Figma**: As the design reference.

## Outcome

Successfully built a pixel-perfect website from Figma design, improving my skills in design-to-code workflow.
