## Overview

This is a collection of multiple Frontend Mentor challenges that I completed to sharpen my HTML, CSS, and JavaScript skills. Each challenge focuses on a specific UI component or layout, helping me practice pixel-perfect design implementation and responsive web development.

## Challenges Completed

- **Profile Card Component**: A clean user profile card with avatar, name, location, and stats.
- **Four Card Feature Section**: A responsive grid layout showcasing four feature cards.
- **NFT Preview Card**: A dark-themed NFT card with hover effects and clean typography.
- **Product Preview Card**: A product page with image, description, pricing, and add to cart button.
- **Space Tourism Preview**: A responsive preview of the space tourism homepage.

## Key Features

- **Pixel-Perfect Design**: Closely matched the provided design files for each challenge.
- **Fully Responsive**: All components optimized for desktop, tablet, and mobile.
- **Clean Code**: Reusable and well-structured HTML and CSS.
- **Hover Effects**: Smooth interactive states and transitions.

## Technologies Used

- **HTML5**: For semantic and structured markup.
- **CSS3**: For styling, layouts, and animations.
- **JavaScript**: For interactive components.
- **Flexbox & Grid**: For modern responsive layouts.

## Outcome

Completing these challenges helped me build strong fundamentals in HTML, CSS, and responsive design. I gained confidence in translating design files into working, production-ready UI components.
